Meshtastic SendFile

sender:
  • compress file <50 kilobytes
  • rename file to "sendfile.jpg" (or .txt or .zip)
  • upload file "sendfile.jpg" to node
  • direct message "sendfile:sendfile.jpg" to receiver
  • get message "ok" from receiver
  • (... send file sendfile.jpg to receiver ...)
  • get message "finished sendfile.jpg" from receiver
receiver:
  • get message "sendfile:sendfile.jpg" from sender
  • direct message "ok" to sender
  • (... get file sendfile.jpg from sender ...)
  • get message "download http://meshtastic.local/sendfile.jpg" from sender
